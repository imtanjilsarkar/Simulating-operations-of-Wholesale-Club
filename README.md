# Wholesale Club Simulation System

A JavaFX-based application simulating the operations of a Wholesale Club (HyperMarket).  
Supports multiple user roles with specific functionalities including HR, Customer Support, Inventory, Cashier, Store, and Membership Management.

## üßë‚Äçüíº Users & Roles

1. **Cashier**  
   - Process customer checkout  
   - Apply membership discount  
   - Process payment  
   - Handle returns/refunds  
   - Generate daily sales report  
   - Balance cash drawer  
   - Manage queue  
   - Flag suspicious activity  

2. **Inventory Manager**  
   - Monitor stock levels  
   - Raise purchase requests  
   - Receive supplier deliveries  
   - Update damaged/expired goods  
   - Forecast demand  
   - Generate stock report  
   - Organize warehouse  
   - Handle supplier returns  

3. **Store Manager**  
   - Oversee daily operations  
   - Approve purchase orders  
   - Manage inventory & staff activities (other goals to be completed)  

4. **Membership Manager**  
   - Register new memberships  
   - Renew memberships  
   - Manage membership database (other goals to be completed)  

5. **HR Manager**  
   - Recruit, remove, update employee info  
   - Approve leave requests  
   - Manage attendance, payroll  
   - Assign work shifts  
   - Generate HR summary report  

6. **Customer Support Staff**  
   - Add, update, close complaints  
   - Check customer order status  
   - Register new memberships  
   - Update customer info  
   - View feedback & generate support report  

7. **Delivery Coordinator**  
   - Assign delivery tasks  
   - Track deliveries  
   - (Other goals can be added)  

8. **Procurement Officer**  
   - Review purchase requests  
   - Create supplier orders  
   - (Other goals can be added)
